! --- YouTube Anti-Adblock Bypass ---
! uBlock Origin patches YouTube at the JS level via trusted scriptlets
! (trusted-replace-fetch-response, trusted-prevent-dom-bypass, trusted-rpnt).
! These scriptlets rewrite YouTube's player response so it never serves ads —
! detection is impossible when the bypass runs cleanly.
!
! Without any intervention, EasyList / AdGuard / OISD / Hagezi sources each
! contribute hundreds of youtube.com-specific cosmetic rules (youtube.com##.ad-*)
! and non-trusted scriptlets. YouTube's DOM mutation observer detects those hidden
! ad elements and triggers the "disable your adblocker" popup even while uBlock's
! bypass is active.
!
! Three-layer fix:
!
! Layer 1 (build-time, primary): scripts/merge.sh strips ALL youtube.com##
!   cosmetic and non-trusted-scriptlet rules from merged sources before output.
!   Only trusted-* bypass scriptlets (e.g. trusted-replace-fetch-response) are
!   preserved. This removes the ~800+ conflicting rules at the source.
!
! Layer 2 (runtime): $genericblock + $generichide disable any remaining generic
!   rules (rules without a youtube.com domain specifier) that would otherwise
!   apply to youtube.com pages — e.g. generic ##.ad-container or OISD's plain
!   ||doubleclick.net^ block.
@@||youtube.com^$genericblock
@@||m.youtube.com^$genericblock
@@||youtube.com^$generichide
@@||m.youtube.com^$generichide
!
! Layer 3 (runtime): Explicitly un-block ad infrastructure domains on youtube.com.
!   YouTube checks these domains to confirm its ad system is reachable. The bypass
!   scriptlet makes YouTube never actually request ads, but if these domains are
!   hard-blocked before the scriptlet runs, it can break the bypass mechanism.
!   Using $important to override any other blocking rules.
@@||googleads.g.doubleclick.net^$domain=youtube.com,important
@@||securepubads.g.doubleclick.net^$domain=youtube.com,important
@@||pubads.g.doubleclick.net^$domain=youtube.com,important
@@||static.doubleclick.net^$domain=youtube.com,important
@@||pagead2.googlesyndication.com^$domain=youtube.com,important
@@||imasdk.googleapis.com^$domain=youtube.com,important
@@||s0.2mdn.net^$domain=youtube.com,important
@@||googlevideo.com^$domain=youtube.com,important
@@||www.youtube.com/api/stats/ads$xhr,domain=youtube.com,important
@@||www.youtube.com/pcs/activeview$xhr,domain=youtube.com,important
@@||www.youtube.com/pagead/^$domain=youtube.com,important
@@||www.youtube.com/get_video_info$xhr,domain=youtube.com,important
